# Introduction

## What is ICAI CAGPT?

ICAI CAGPT is an **AI-powered accounting and tax assistant** that helps:
- 💼 **Business owners** manage finances
- 📊 **Accountants** automate calculations
- 📋 **CPAs** create professional documents
- 🔍 **Auditors** detect fraud and anomalies

## Key Features

| Feature | Description |
|---------|-------------|
| **AI Chat** | Ask any accounting or tax question |
| **10 Expert Modes** | Specialized AI for different tasks |
| **Document Analysis** | Upload and analyze financial documents |
| **Export Options** | Download as PDF, Word, Excel, PowerPoint |
| **Multi-Jurisdiction** | USA, Canada, UK, India tax support |

# Getting Started

## For All Users: How to Login

### Step 1: Open the Login Page
1. Open your web browser
2. Go to: **https://cagpt.icai.org/auth**

### Step 2: Enter Your Credentials
1. Type your **Email** in the email field
2. Type your **Password** in the password field

### Step 3: Click Sign In
1. Click the purple **"Sign In"** button
2. Wait for the page to load
3. ✅ You will see the Chat page

### What If Login Fails?
| Error | Solution |
|-------|----------|
| "Incorrect password" | Check spelling, passwords are case-sensitive |
| "User not found" | Check email spelling or create account |
| "Account locked" | Wait 15 minutes or reset password |

---

## For New Users: How to Create Account

### Step 1: Go to Sign Up
1. Go to: **https://cagpt.icai.org/auth**
2. Click **"Sign up"** link below the login form

### Step 2: Fill the Registration Form
1. **Full Name**: Enter your full name
2. **Email**: Enter a valid email address
3. **Password**: Create a strong password (minimum 8 characters)

### Step 3: Create Your Account
1. Click **"Create Account"** button
2. Check your email inbox
3. Click the verification link in the email
4. ✅ Account created! You can now login
